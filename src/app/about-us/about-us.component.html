Tentang Kami

Untuk keperluan ujian akhir semester mata kuliah IF231 - Pengantar Teknologi Internet, kami membuat sebuah website peta menggunakan framework Angular dan API ArcGIS untuk mengembangkan dan menyelesaikan proyek Ujian Akhir Semester kami. Dalam website kami, ada beberapa fitur-fitur menarik yang kami buat menggunakan API ArcGIS seperti fitur Zoom In/Zoom Out, Fitur mencari lokasi, Fitur bookmark, Fitur mengganti desain peta, dan banyak lainnya.
